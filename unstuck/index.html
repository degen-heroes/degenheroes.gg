Find Stuck Heroes on CrystalVale Quests

Made by thanpolas - source on Github

This tool will help you check if an address has stuck heroes. If you are checking your own address, and you have stuck heroes, there will be an option to get them unstuck, by finishing the quest. But most likely it will not work.

Please switch to DFK Chain Network!!!


WARNING - READ Carefully

This button will clean all your clears all quests from all queues, then goes through and checks every Hero the player owns. If the Hero is set to questing, it will clear them out of that and set their stamina to 0.

So make sure your wallet only has stuck quests so your normal ones will not be terminated prematurely
